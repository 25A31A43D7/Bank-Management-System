# Bank Management System in C++

## Overview
This project is a simple **Bank Management System** developed using **C++** and **Object-Oriented Programming (OOP)** concepts.  
It allows users to create a bank account, deposit money, withdraw money, check balance, and display account details through a menu-driven program.

---

## Features
- Create a new bank account
- Deposit money
- Withdraw money
- Check account balance
- Display account details
- Menu-driven user interface

---

## Technologies Used
- C++
- OOP Concepts
- Functions
- Classes and Objects

---

## Class Used

### `BankAccount`
The `BankAccount` class contains:
- Account Holder Name
- Account Number
- Account Balance

### Member Functions
| Function | Description |
|----------|-------------|
| `createAccount()` | Creates a new account |
| `deposit()` | Deposits money into the account |
| `withdraw()` | Withdraws money from the account |
| `checkBalance()` | Displays current balance |
| `displayDetails()` | Displays account details |

---

## Program Flow
1. User selects an option from the menu.
2. Required operation is performed.
3. Menu appears again until the user selects Exit.
